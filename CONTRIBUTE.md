## How to contribute

### Install
1. `yarn`
1. `yarn link`
1. `yarn start`
1. `cd example` in a new terminal window
1. `yarn`
1. `yarn link react-router4-with-layouts`
1. `yarn start` 

### Pull request
1. Respect the eslint rules
1. Commit using `yarn cm` command
1. Add tests if possible (currently not)
